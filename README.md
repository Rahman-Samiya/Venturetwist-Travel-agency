# VentureTwist - Travel & Adventure Website

A comprehensive travel and adventure booking website featuring destinations, activities, travel gadgets, and more.

## Project Overview

VentureTwist is a full-featured travel website that allows users to explore destinations, book adventure activities like scuba diving, sky diving, and bungee jumping, and purchase travel gadgets.

## Features

### Destinations
- **Maldives** - Tropical beach paradise
- **Nepal** - Himalayan adventures
- **Thailand** - Cultural exploration
- **Singapore** - Modern city experiences

### Adventure Activities
- **Scuba Diving** - Explore underwater world with SCUVA Diving
- **Sky Diving** - Experience thrill of freefall
- **Bungee Jumping** - Leap of faith at various locations

### Travel Gadgets
- Backpacks
- Life Jackets
- Travel Shoes
- Power Banks
- Travel Accessories

### Additional Features
- Hotel Booking System
- Transport Booking
- Travel Diaries (User Stories)
- User Authentication (Login/Register)

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Styling and responsive design
- **JavaScript** - Interactive features
- **PHP** - Server-side processing
- **MySQL** - Database (wanderful.sql)

## Getting Started

1. Clone or download the project
2. Set up a local server (XAMPP, WAMP, or similar)
3. Import the database from `wanderful.sql`
4. Configure database connection in PHP files
5. Open `index.html` in your browser
